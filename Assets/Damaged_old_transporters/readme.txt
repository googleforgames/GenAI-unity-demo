This package contains 2 different types old transporters and both in 3 levels of damage.

Just drag the prefabs into your scene and arrange them as you like.

You may also want to have a look at the example scene that contains all the prefabs.

All single objects using between 3300 und 4700 polygons.

For both transporters the package contains 12 different texture-sets.

Textures-sizes 2048x2048.

The original FBX-file is also included in the package.
